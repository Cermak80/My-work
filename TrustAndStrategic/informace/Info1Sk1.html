Instrukce

V experimentu vám bude náhodně přidělena role hráče A, B nebo C. Instrukce na této obrazovce jsou stejné pro hráče v každé roli. V každém kole budete hrát hru, jejíž průběh bude následující:

{{ if Treatment == "1" or Treatment == "2"}}
1. Hráč B se závazně rozhodne kolik procent z částky získané po ztrojnásobení vrátí hráči A. Zbylou částku si ponechá. Kolik procent pošle zpět vidí hráč C, ale nikoli Hráč A.
{{ endif }} {{ if Treatment == "3" or Treatment == "4"}}
1. Hráč B se závazně rozhodne kolik procent z částky získané po ztrojnásobení vrátí hráči A. Zbylou částku si ponechá. Kolik procent pošle zpět nevidí ani hráč C ani hráč A.
{{ endif }}
2. Hráč B pošle hráči A informaci, kolik procent zvolil. Hráč B se však nemusí rozhodnout poslat informaci o skutečně zvolené částce. Hráči A se tato informace zobrazí jako procento poslané čásky.

Příklady:
Hráč B se rozhodne říci, že vrátí 50 % toho, kolik po ztrojnásobení od hráče A dostane. Hráč A tedy na obrazovce uvidí, že se mu vrátí 150 % poslané částky.

Hráč B se rozhodne říci, že vrátí 30 % toho, kolik po ztrojnásobení od hráče A dostane. Hráč A tedy na obrazovce uvidí, že se mu vrátí 90 % poslané částky.

Hráč B se rozhodne říci, že vrátí 0 % toho, kolik po ztrojnásobení od hráče A dostane. Hráč A tedy na obrazovce uvidí, že se mu vrátí 0 % poslané částky.

{{ if Treatment == "1" or Treatment == "2"}}
3. Hráč C pošle hráči A informaci, kolik procent hráč B zvolil. Ani hráč C nemusí poslat informaci o skutečně zvolené částce. Hráči A se tato informace opět zobrazí jako procento poslané částky.
{{ endif}} {{ if Treatment == "3" or Treatment == "4"}}
3. Hráč C pošle hráči A informaci, kolik procent hráč B zvolil. Hráč C ovšem ve skutečnosti neví, jaké procento hráč B zvolil. Hráči A se tato informace opět zobrazí jako procento poslané částky.
{{ endif }}
4. Hráč A si zvolí kolik Kč pošle hráči B.

5. Transakce mezi hráči A a B se vyhodnotí.
  • Hráč A získá částku, kterou mu poslal zpět hráč B a částku, kterou neposlal hráči B
  • Hráč B získá trojnásobek částky poslané hráčem A minus procentuální podíl této částky, který se rozhodl poslat zpět.
    • {{ if Treatment == "1" or Treatment == "3"}}
  • Hráč C získá fixní bonus 50 Kč.
    • {{ endif }} {{ if Treatment == "2" or Treatment == "4"}}
  • Odměna hráče C je tím menší, čím více pošle hráč A hráči B. Konkrétně, hráč C získá 100 Kč minus částku, kterou poslal hráč A hráči B.
    • {{ endif }}

Informaci o vydělané částce zjistíte až na úplném konci druhého experimentu.
